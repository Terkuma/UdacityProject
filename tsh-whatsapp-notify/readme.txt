Professional WhatsApp notification system for WooCommerce. Send automated order and customer messages via the Meta Cloud API.

== Description ==

TSH WhatsApp Notify integrates your WooCommerce store with the Meta WhatsApp Cloud API to deliver real-time automated notifications to both store admins and customers.

= Key Features =

* Automated order status notifications via WhatsApp
* Customisable message templates with dynamic order variables
* Smart message queue with retry logic
* Full message logging and history
* Admin notifications for new orders, payment failures, low stock
* Customer notifications for order placement, shipping, delivery

= Requirements =

* WordPress 6.3+
* WooCommerce 7.0+
* PHP 8.1+
* Meta WhatsApp Business Account (Phase 2)

== Installation ==

1. Upload the plugin to the `/wp-content/plugins/` directory.
2. Activate the plugin through the **Plugins** menu.
3. Navigate to **WooCommerce → TSH WhatsApp Notify** to configure the plugin.

== Frequently Asked Questions ==

= Does this plugin work without WooCommerce? =

No. TSH WhatsApp Notify requires WooCommerce to be installed and active.

= Which PHP version is required? =

PHP 8.1 or higher.
